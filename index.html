Recursive Descent Parser By: Vlad Marinov

  BNF Grammar:

Example Of Valid String

1+2$
0-3*2$
2/(3+1)$

Example Of Invalid String

1+2
1*4$
1*a$

EXP ::= EXP + TERM | EXP - TERM | TERM
TERM ::= TERM * FACTOR | TERM / FACTOR | FACTOR
FACTOR ::= ( EXP ) | DIGIT
DIGIT ::= 0 | 1 | 2 | 3

Enter any string containing the numbers 0, 1, 2 or 3 and the symbols +, -, *, /, (, and ). And string should end with dollar sign ($). When you click on 'Validate' button, it will check whether the string provided is legal or not for a given BNF grammar.
